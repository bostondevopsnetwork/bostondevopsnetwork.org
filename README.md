# bostondevopsnetwork.org

This is the Jekyll site behind
[bostondevopsnetwork.org](https://bostondevopsnetwork.org), which is
hosted by Github Pages.

## Testing Changes

Assuming that you already have [Docker](https://www.docker.com/)
installed and running, simply run:

```
./jekyllserver.sh
```

...then navigate to `http://localhost:4000` in your browser.
